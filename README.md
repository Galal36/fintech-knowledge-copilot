# FinTech Knowledge Copilot

AI Knowledge Operations System for multi-source document ingestion, retrieval-augmented question answering, and proactive insights generation.

## Planned Stack

- Backend: FastAPI, SQLAlchemy, PostgreSQL
- Vector Store: ChromaDB
- AI: Gemini API
- Frontend: Next.js, Tailwind CSS
- Infra: Docker Compose

## Repository Structure

```text
backend/   FastAPI application, business logic, tests
frontend/  Next.js application
infra/     Local infrastructure configuration
docs/      Supporting technical documentation
```

## Status

Initial repository scaffold in place. Backend, frontend, and infrastructure implementation will be added incrementally.
